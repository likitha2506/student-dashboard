# Student Dashboard

## 📌 Project Description
This project is a student dashboard that manages student data.

## 🚀 Features
- View student details
- Manage database
- User-friendly interface

## 🛠️ Technologies Used
- Python
- Database (SQLite / MySQL)
- Streamlit / Flask (if used)

## ▶️ How to Run
1. Install requirements:
   pip install -r requirements.txt
2. Run the app:
   python app.py
